## MinsaPay

민족제(민사고 여름 축제)를 위한 결제 서버와 웹 앱입니다.

Payment Server and Payment Web App for [KMLA](https://en.wikipedia.org/wiki/Korean_Minjok_Leadership_Academy) Minjok Festival (Summer Festival)
